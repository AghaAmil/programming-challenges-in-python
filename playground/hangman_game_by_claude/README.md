## How to Play

1. The game randomly selects a word
2. You guess one letter at a time
3. Correct guesses reveal the letter in the word
4. Wrong guesses add to the hangman drawing
5. Win by guessing the word before the hangman is complete!
6. You have 6 wrong attempts before losing

## Running the Game

```bash
# Navigate to the game directory
cd hangman_game_by_claude

# Run the game
python hangman.py
```

Or run directly from the project root:

```bash
python hangman_game_by_claude/hangman.py
```

## Game Rules

- Enter one letter at a time
- Letters are case-insensitive
- Already guessed letters will be tracked
- You can see all your previous guesses
- 6 wrong attempts allowed
